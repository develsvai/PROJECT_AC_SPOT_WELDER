AC Spot Welder Simulator

Event Log

최신 로그가 위에 쌓입니다. 자동 스팟은 보통 `접촉 감지 -> 대기 -> 펄스 출력 -> 완료 -> lockout` 순으로 보면 됩니다.

Controls

Manual Weld
수동 모드 전용입니다. 이제 시뮬레이터에서는 버튼을 누르면 필요한 시간까지 자동으로 흘려서 결과가 바로 보입니다.
Contact Input
AUTO 모드에서는 `Touch Pulse`가 가장 직관적입니다. 접촉을 한번 만들고 해제해서 자동 감지와 용접을 한 사이클로 보여줍니다.
Simulation Tools
`Warm Up 1s`는 부팅 직후 splash를 넘기고 auto detect baseline을 안정시키는 용도입니다. `Reset`은 초기 상태로 되돌립니다.
